Doing Tables

Define a table by the class='example'><table> tag.


<tr> defines rows, <th> is a column heading, <td> are the individual cells

The regular CSS border property defines the table and table cell borders. Setting {border-collapse: collapse} will collapse the borders into one.

<table style="width:100%">
<tr>
<th>Firstname</th>
<th>Lastname</th>
<th>Age</th>
</tr>
<tr>
<td>Jill</td>
<td>Smith</td>
<td>50</td>
</tr>
<tr>
<td>Eve</td>
<td>Jackson</td>
<td>94</td>
</tr>
</table>



the code above results in:

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94
