Transition Properties change values over a Specified Duration

In this example, the class .transition-example and .transition-example:hover have different properties. Note the last property in the css.

.transition-example{ width:75%; margin:2em auto; padding:1em; border:3px groove gainsboro; transition: all 0.5s ease; }
Transition property values:
  1. The first shorthand value is Transition-Property. This is the property that should be changed.
  2. The next value is Transition-Duration. In seconds, how long the element should take to change.
  3. The third value is Transition-Timing-Function. This specifies the animation curve for intermediary values.
