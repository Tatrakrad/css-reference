This is a CSS Web Font

With the magic of CSS3 and Content Delivery Networks

You can foist whatever asinine typography you wish upon the hapless internet yeomanry.

You can use fonts other than the 'web-safe' choices by using one or more of these handy CSS rules.

  • @font-face
  • @import

@font-face lets you specify a source for a font-file, or sources for fonts in multiple formats, and assign them names to be used in the CSS 'Font-Family' rules

  • @font-face {
  • font-family: "font of all knowledge";
  • src: url(fontofallknowledge.woff);
  • }

@import is used to extend your stylesheet, and is the easiest way to add google fonts to your webzone.

the example below imported the 'Shrikhand' and 'Yantramanav' fonts into this page.

  • @import
  • url('https://fonts.googleapis.com/css?family=Shrikhand|Yantramanav');
